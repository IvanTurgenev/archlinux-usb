Install Sublime Text editor (not available in official Arch repository)
https://www.sublimetext.com/docs/3/linux_repositories.html

Install the GPG key:
curl -O https://download.sublimetext.com/sublimehq-pub.gpg && sudo pacman-key --add sublimehq-pub.gpg && sudo pacman-key 
--lsign-key 8A8F901A && rm sublimehq-pub.gpg

Select the stable channel:
echo -e "\n[sublime-text]\nServer = https://download.sublimetext.com/arch/stable/x86_64" | sudo tee -a /etc/pacman.conf

Update pacman and install Sublime Text:
sudo pacman -Syu sublime-text
